# 🛠️ I Need - Profesionales para tu hogar

**I Need** es una landing page moderna y minimalista diseñada para conectar a usuarios con profesionales verificados en Cataluña. Desde cerrajería y fontanería hasta reformas integrales, la plataforma facilita la gestión de servicios domésticos con confianza y rapidez.

![Vista previa del proyecto](apple-touch-icon.png)

## ✨ Características principaless

* **Diseño Premium:** Interfaz limpia utilizando una paleta de colores crema (`#fafaf9`) y tipografía elegante (*DM Sans* e *Instrument Serif*).
* **Totalmente Responsivo:** Optimizado para dispositivos móviles, tablets y escritorio.
* **Formulario Inteligente:** Sistema de solicitud de servicios integrado con validación.
* **Selección Rápida:** Las tarjetas de servicios (Cerrajería, Fontanería, etc.) seleccionan automáticamente la opción correcta en el formulario al hacer clic.
* **Optimización SEO:** Metatags configurados para mejorar la visibilidad en buscadores.

## 🚀 Tecnologías utilizadas

Este proyecto fue construido buscando la máxima velocidad de carga sin dependencias pesadas:

* **HTML5** - Estructura semántica.
* **Tailwind CSS** - Estilizado rápido y moderno mediante CDN.
* **JavaScript (Vanilla)** - Lógica de interacción, animaciones de scroll y gestión de modales.
* **Google Fonts** - Tipografías personalizadas para un acabado profesional.
